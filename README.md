# Velour Restaurant Booking System

Система бронирования столиков для ресторана высокой кухни Velour.

## О проекте

Velour — это элегантный ресторан высокой кухни с камерной атмосферой, где каждое блюдо создаётся из отборных продуктов, а каждый вечер — как отдельная история. Проект включает систему онлайн-бронирования с современным дизайном и интуитивным интерфейсом.

### Особенности

- **Современный дизайн**: Элегантный интерфейс с использованием CSS Modules и TypeScript
- **Адаптивная верстка**: Оптимизирован для всех устройств
- **Система бронирования**: Удобная система выбора столиков и времени
- **Анимации**: Плавные переходы и микро-взаимодействия
- **База данных**: Интеграция с Supabase для управления бронированиями

## Технологический стек

- **Frontend**: Next.js 16.1.6, React 19.2.3
- **Стили**: CSS Modules, Tailwind CSS
- **Типизация**: TypeScript
- **База данных**: Supabase
- **Шрифты**: Cormorant Garamond, Cormorant SC

## Начало работы

### Установка зависимостей

```bash
npm install
```

### Переменные окружения

Создайте файл `.env.local` с следующими переменными:

```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### Запуск разработки

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

### Сборка проекта

```bash
npm run build
npm start
```

## Доступные скрипты

- `npm run dev` - Запуск сервера разработки
- `npm run build` - Сборка проекта для продакшена
- `npm start` - Запуск продакшн-сервера
- `npm run lint` - Проверка кода ESLint
